% Métricas de evaluación
\section{Evaluación}

\begin{frame}[t,plain]
	\vspace{0.4cm}
	\begin{flushleft}
		{\Large\bfseries Métricas de evaluación}
	\end{flushleft}

	\vspace{0.6cm}
	\begin{itemize}
		\item Log-likelihood, AIC y BIC para evaluar el ajuste del modelo.
		\item Precisión de clasificación (consumo vs. no consumo).
		\item Matriz de confusión para errores entre niveles de consumo.
		\item Curva ROC y AUC para evaluar la ecuación de participación.
		\item Comparación con modelos de ML manteniendo coherencia con la interpretación económica.
	\end{itemize}

	\vspace{0.4cm}
	{\small Estas métricas permiten valorar el poder explicativo y predictivo del modelo sin perder de vista el marco teórico.}

	\vfill
\end{frame}
