\section{Metodología}

\begin{frame}[t,plain]
	\vspace{0.4cm}
	\begin{flushleft}
		{\Large\bfseries Variables}
	\end{flushleft}

	\vspace{0.6cm}
	{\bfseries Variable objetivo y variables explicativas}

	\vspace{0.4cm}
	{\bfseries Variable dependiente (Y):} nivel de consumo de psilocibina:\\
	0 = no consume \quad 1 = consumo ocasional \quad 2 = consumo frecuente

	\vspace{0.6cm}
	{\bfseries Variables explicativas principales:}
	\begin{itemize}
		\item Política no restrictiva (dummy)
		\item Edad, sexo, educación, ingresos, empleo
		\item Contexto territorial (urbano/rural, región)
		\item Conductas de salud y consumo; salud mental autoinformada
		\item Acceso institucional (programas de reducción de daños, salud mental)
	\end{itemize}

	\vspace{0.4cm}
	{\small Estas variables permiten capturar tanto determinantes económicos como psicosociales del consumo.}

	\vfill
\end{frame}
